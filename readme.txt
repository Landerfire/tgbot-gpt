Реализация nodejs приложения - телеграм-бота с поддержчкой ChatGPT 3.5

Для работы необходимо добавить и настроить config файлы с персональным токеном openai и токеном телеграм бота

Реализованные на данный момент функции:
    > Работа с текстовыми сообщениями
    > Работа с голосовыми сообщениями
    > Сохранение и очистка контекста сессии
